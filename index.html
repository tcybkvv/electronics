SQL Запросы

select c.name as client_name, i.id as invoice_id, i.date, i.total_amount from clients c left join invoices i on c.id = i.client_id;

select p.name, p.quantity, w.location from products p join warehouse w on p.warehouse_id = w.id order by p.quantity desc;

Показывает список товаров с их количеством и местоположением склада.

select name, price, quantity from products where is_online = true;

Выводит онлайн-товары, которые доступны на сайте: название, цену и количество.

namepricequantity
 
select i.id as invoice_id, d.status, d.delivery_date, d.courier_name from invoices i join delivery d on i.id = d.invoice_id where i.delivery_required = true;

Показывает накладные: их статус, дату доставки и имя курьера.

select p.name, sum(ii.quantity) as total_sold from invoice_items ii join products p on ii.product_id = p.id group by p.id order by total_sold desc limit 15;

Выводит топ-15 самых продаваемых товаров по суммарному количеству продаж.

nametotal_sold
 
select e.name, count(i.id) AS sales_count, sum(i.total_amount) as total_sales from employees e join invoices i on e.id = i.employee_id group by e.id;

Показывает, сколько продаж сделал каждый сотрудник и на какую сумму всего.
